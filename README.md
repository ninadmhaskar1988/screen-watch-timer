# Screen Watch Timer ⏱️

[![Firefox Add-on](https://img.shields.github.com/amo/v/screen-watch-timer?color=orange&label=Firefox%20Add-on&logo=firefox-browser)](https://addons.mozilla.org/firefox/addon/screen-watch-timer/)

A professional, distraction-free screen time tracker for Firefox. Designed for focus and eye health.

## 📥 Installation

### Official Store (Recommended)
Install the latest stable version directly from the store:
👉 **[Download on Firefox Add-on Market](https://addons.mozilla.org/firefox/addon/screen-watch-timer/)**

### Developer Mode
1. Clone this repo: `git clone https://github.com/ninadmhaskar1988/screen-watch-timer.git`
2. Open Firefox and go to `about:debugging#/runtime/this-firefox`.
3. Click **Load Temporary Add-on...** and select `manifest.json`.

## ✨ Features
* **Floating "Pill" UI**: A sleek, animated timer overlay that stays visible but non-intrusive.
* **10-Minute Break Alerts**: Automatic reminders to rest your eyes.
* **Smart Session Reset**: Timer resets automatically when the browser restarts.
* **Privacy Focused**: No data collection; all tracking stays local.

## 🛠️ Tech Stack
* **Manifest V3**
* **WebExtensions API** (Storage, Tabs)
* **CSS3** (Backdrop-filter blur, Scale animations)
