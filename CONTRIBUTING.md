# Contributing to Screen Watch Timer ⏱️

First off, thank you for considering contributing! It’s people like you that make this tool better for everyone.

## 🚀 How Can I Help?

### 💡 Suggesting Ideas
Have a cool idea for a new feature? 
1. Check the [Discussions](https://github.com/ninadmhaskar1988/screen-watch-timer/discussions) tab to see if it's already being talked about.
2. If not, start a new discussion!

### 🐛 Reporting Bugs
Found something broken?
1. Open an [Issue](https://github.com/ninadmhaskar1988/screen-watch-timer/issues).
2. Describe what happened and include your Firefox version.

### 👩‍💻 Submitting Code (Pull Requests)
1. Fork the repo and create your branch (`git checkout -b feature/AmazingFeature`).
2. Keep the UI minimalist (stick to the "pill" design).
3. Commit your changes and open a Pull Request!

## 🎨 Design Rules
* **Stay Minimal:** No heavy borders or square corners. Use `border-radius: 100px`.
* **Privacy First:** Never add code that sends data to an external server.
* **Performance:** Ensure animations are smooth and don't lag the browser.

Thank you for helping build the best screen timer for Firefox! ✨
